# أقساط — إدارة الالتزامات المالية (الأقساط العقارية)

تطبيق ويب بسيط لمتابعة الأقساط العقارية المستحقة، صُمِّم لمن اشترى وحدات بالتقسيط
في مصر ويقيم في السعودية. يعرض المبالغ **بالجنيه المصري (ج.م)** مع مقابلها
**بالريال السعودي (﷼)** حسب سعر صرف قابل للتعديل.

## المزايا

- **لوحة تحكم** تعرض: إجمالي المتبقّي، المستحق خلال ٩٠ يوماً، المبالغ المتأخّرة، والقسط القادم.
- **إدارة الوحدات**: أضف كل وحدة عقارية بسعرها الإجمالي والمقدّم ونسبة السداد.
- **الأقساط**: أضف قسطاً مفرداً، أو أنشئ **جدول أقساط كامل** تلقائياً (شهري / ربع سنوي / نصف سنوي / سنوي).
- **العملتان معاً**: كل مبلغ يظهر بالجنيه المصري والريال السعودي.
- **تنبيهات بصرية** للأقساط المتأخّرة والمستحقة قريباً.
- **تصدير / استيراد** البيانات كملف JSON للنسخ الاحتياطي.
- **يعمل دون إنترنت** — البيانات تُحفظ على جهازك فقط (localStorage)، ويمكن تثبيته كتطبيق (PWA).

## طريقة الاستخدام

افتح `index.html` في المتصفح مباشرة، أو استضِفه على أي خدمة استضافة ثابتة
(مثل GitHub Pages). لا يحتاج أي خادم أو تثبيت.

### سعر الصرف
عدّل قيمة «١ ريال = … ج.م» في أعلى الشاشة، وستتحدّث كل المبالغ بالريال فوراً.

## الخصوصية
كل بياناتك تبقى داخل متصفحك ولا تُرسَل إلى أي خادم. استخدم زر «تصدير البيانات»
لعمل نسخة احتياطية بشكل دوري.

## البنية التقنية
- HTML/CSS/JavaScript خالص بدون أي مكتبات خارجية.
- `index.html` — الواجهة، `styles.css` — التنسيق، `app.js` — المنطق، `manifest.json` — إعداد PWA.
